Projeto Front End

Siga Minhas Redes Sociais

Desenvolvi um projeto no qual utilizei o Figma para criar um design inicial e, em seguida, implementei o layout usando HTML e CSS. Comecei pelo Figma, onde esbocei a estrutura do site, incluindo os elementos de navegação, seções de conteúdo e áreas interativas. No Figma, trabalhei com as cores, fontes e espaçamento para garantir uma experiência visual atraente e coerente. Usei as funcionalidades de prototipagem do Figma para criar interações básicas e testar a navegação, permitindo ajustes finos antes de iniciar a codificação. Com o design finalizado, passei para a implementação em HTML e CSS. Estruturei o HTML de forma semântica, usando tags adequadas para cada parte do conteúdo, garantindo acessibilidade e facilidade de manutenção. No CSS, utilizei técnicas para criar um layout responsivo que se adapta bem a diferentes tamanhos de tela. O resultado foi um projeto coeso e funcional, alinhando design e desenvolvimento de maneira harmoniosa.

Um Pouco Mais Sobre

Concluindo meu estudo sobre o desenvolvimento de projetos utilizando Figma e as linguagens HTML e CSS, percebi a importância de uma abordagem integrada entre design e codificação. O uso do Figma permite uma visualização clara e detalhada do projeto desde suas fases iniciais. A transição para HTML e CSS, por sua vez, revela a importância de uma estrutura semântica e estilização eficiente para a criação de interfaces responsivas e interativas. A combinação de prototipagem no Figma com técnicas avançadas de CSS, resulta em um produto final que não só atende às expectativas visuais, mas também proporciona uma experiência de usuário fluida e agradável. Esse estudo reforçou a necessidade de equilíbrio entre design criativo e codificação técnica, elementos fundamentais para o desenvolvimento de projetos web eficazes e inovadores.
